Storage as a Service (STaaS)

Huawei STaaS solution is an on-demand solution that enables automated storage services provisioning and intelligent data management in enterprise data centers. This solution allows for cloud-based critical services and helps transform traditional data centers to cloud-based ones.

Converge Data Management (CDM)

Huawei CDM solution uses storage features and the unified user interface to implement converged management of disaster recovery, backup, and data reuse to reduce construction costs and improve data values.

FusionROBO

Huawei FusionROBO is an integrated solution designed for enterprise branches, which integrates the servers, storage, switches, firewalls, routers, and uninterruptible power supplies (UPSs) required by branches. It is delivered in a cabinet and can be deployed in the office environment through FusionROBO Manager to implement remote centralized management. It is an ideal choice for branch informatization.
