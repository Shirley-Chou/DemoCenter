Advantages

  • Core Service Bearing
    Provides ultimate performance and reliability for ission-critical applications.
    • Ultimate performance:
      The distributed storage system based on Flash Native adopts innovative Huawei FlashLinkTM technology to provide 140,000 IOPS@1 ms/node performance, ranking No. 1 in the industry.
    • Rock-solid reliability:
      Local protection: Millisecond-level node fault detection and five-second switchover
      Remote DR: Gateway-free distributed active-active, RPO = 0, RTO ≈ 0, providing 99.9999% availability
  • Storage Consolidation
    Flexible, efficient, and open ompatibility support smooth cloud evolution of enterprise data centers
    • Open ecosystem：
      Supports seamless access to containers, VMs, general-purpose servers, and UNIX servers, simplifying the data center storage architecture.
    • Supports mainstream hypervisors such as vSphere, FusionSphere, Hyper-V, OpenStack, KVM, and Xen architecture without platform binding.
  • Hybrid Cloud Support
    Unified architecture, unified services, and unified APIs help smooth cloud migration for services.
    • Native hybrid cloud backup：
      Based on the backup function of the storage system, the third-party backup software is not required for cross-cloud data protection.
    • Native hybrid cloud DR：
      Based on the asynchronous replication technology between private cloud and public cloud storage, the gateway-free cross-cloud service continuity is achieved.

Application Scenario

R&D Cloud

FusionStorage cloud storage provides elastic on-demand resource deployment and simple management for frequently updated development and testing environments, allowing users to quickly develop and launch innovative products.

Government Cloud

FusionStorage cloud storage builds a converged storage pool for government clouds to share data sets and one-stop storage services, replacing siloed resource pools, reducing cross-pool data copy, and improving data service efficiency.

Operation Cloud

Uses FusionStorage to build a massive cloud storage system that can be linearly expanded on demand to support carriers' flexible expansion. Supports content operations and self-operation service systems rollout such as OTT and IPTV.
